Air Canvas

Air canvas helps to draw on a screen just by waiving your finger in the air. We will be using the computer vision techniques of OpenCV to build this project. The preferred language is python due to its exhaustive libraries and easy to use syntax but understanding the basics it can be implemented in any OpenCV supported language. We will be using a colored object at tip of finger as the marker
